# Cadastro de Carros

Sistema de cadastro de carros simples.

## 🧠 Conhecimento utilizados
- Java
- POO
- Listas e Coleções

## 📋 Funcionalidades
- Cadastrar carros
- Listar os carros cadastrados
- Remover carros
- Encerrar o programa

## ▶️ Como executar
1. Clone o repositório
2. Abra no Eclipse
3. Execute a classe Main
